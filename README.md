## ✨ Features

- 🌗 **Dark, Light, Toggle, or Custom color mode** — set it in the config
- 🔗 **Unlimited links** — add as many as you need
- 👤 **Avatar support** — use a photo URL or fall back to initials
- 📱 **Fully responsive** — looks great on mobile and desktop
- ⚡ **Zero dependencies** — pure HTML, CSS, and JS
- 🛠️ **One config block** — no hunting through code

## 🚀 Quick Start

```bash
git clone https://github.com/bloxiebuilds/linknest-Link-In-Bio-Template.git
cd linknest-Link-In-Bio-Template
```

Open `index.html`, scroll to the `CONFIG` block at the top, and fill in your details. Done!
